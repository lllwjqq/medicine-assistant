# 智能药盒 API 接口文档

## 一、通用说明

| 项目 | 说明 |
|------|------|
| 项目类型 | uni-app 微信小程序 |
| 接口根路径 | `/api/medicineBox` |
| 请求格式 | `application/json` |
| 身份验证 | 除登录接口外，所有请求头必须携带 `token`（登录令牌） |

### 统一返回格式

```json
{
  "code": 200,
  "msg": "提示信息",
  "data": {}
}
```

### 状态码说明

| code | 说明 |
|------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未授权 / token 无效 |
| 500 | 服务器内部错误 |

## 二、用户模块

### 1. 微信授权登录

| 项目 | 内容 |
|------|------|
| 接口地址 | `/api/medicineBox/user/login` |
| 请求方式 | `POST` |
| 请求头 | `Content-Type: application/json`（无需 token） |

**请求体**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| code | String | 是 | 微信登录临时 code（`wx.login` 获取） |

**返回示例**

```json
{
  "code": 200,
  "msg": "登录成功",
  "data": {
    "userId": "10001",
    "nickName": "张三",
    "avatar": "https://xxx.com/avatar.jpg",
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
  }
}
```

### 4. 保存用药信息

| 项目 | 内容 |
|------|------|
| 接口地址 | `/api/medicineBox/ocr/save` |
| 请求方式 | `POST` |
| 请求头 | `token` |

**请求体**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| userId | String | 是 | 用户唯一 ID |
| medicineName | String | 是 | 药品名称（支持手动修正） |
| dailyTimes | Number | 是 | 每日服用次数（正整数） |
| dose | String | 是 | 每次用量（如「1 片」「5ml」） |
| takeTime | String | 是 | 服用时机（饭前 / 饭后 / 空腹 / 睡前） |
| cycle | Number | 是 | 服药周期（天，默认 7） |

**返回示例**

```json
{
  "code": 200,
  "msg": "保存成功",
  "data": {
    "medicineId": "20001"
  }
}
```

## 四、服药提醒模块

### 5. 添加服药提醒

| 项目 | 内容 |
|------|------|
| 接口地址 | `/api/medicineBox/reminder/add` |
| 请求方式 | `POST` |
| 请求头 | `token` |

**请求体**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| userId | String | 是 | 用户唯一 ID |
| medicineId | String | 是 | 药品 ID |
| remindTimes | Array\<String\> | 是 | 提醒时间列表（如 `["08:00","12:00"]`） |
| repeatType | String | 是 | 重复类型：`daily`（每天）/ `weekday`（工作日） |
| status | String | 否 | 提醒状态：`enable`（启用）/ `disable`（暂停） |

**返回示例**

```json
{
  "code": 200,
  "msg": "添加提醒成功",
  "data": {
    "reminderId": "30001"
  }
}
```

---

## 五、服药记录模块

### 9. 标记已服药

| 项目 | 内容 |
|------|------|
| 接口地址 | `/api/medicineBox/record/add` |
| 请求方式 | `POST` |
| 请求头 | `token` |

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| userId | String | 是 | 用户唯一 ID |
| medicineId | String | 是 | 药品 ID |
| planTime | String | 是 | 计划服用时间（如 `"2026-02-24 08:00:00"`） |
| actualTime | String | 是 | 实际服用时间（如 `"2026-02-24 08:05:00"`） |

**返回示例**

```json
{
  "code": 200,
  "msg": "标记成功",
  "data": {
    "recordId": "40001"
  }
}
```

---

### 10. 获取服药记录

| 项目 | 内容 |
|------|------|
| 接口地址 | `/api/medicineBox/record/list` |
| 请求方式 | `GET` |
| 请求头 | `token` |

**请求参数（Query）**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| userId | String | 是 | 用户唯一 ID |
| startTime | String | 是 | 开始时间（如 `"2026-02-01 00:00:00"`） |
| endTime | String | 是 | 结束时间（如 `"2026-02-28 23:59:59"`） |

**返回示例**

```json
{
  "code": 200,
  "msg": "查询成功",
  "data": [
    {
      "recordId": "40001",
      "medicineId": "20001",
      "medicineName": "布洛芬缓释胶囊",
      "planTime": "2026-02-24 08:00:00",
      "actualTime": "2026-02-24 08:05:00",
      "status": "taken"
    }
  ]
}
```
